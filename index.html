Web Application

Dykstra's Algorithm Visualiser

This is a project that take Dykstra's algorithm and applies it to a maze. The maze is 15 x 30 and the algorithm is applied to find the shortest path from the start to the end. The algorithm is visualized in the browser and the shortest path is highlighted in yellow. In this, the user can select the start and end points, as well as the walls that will be used to create the maze. Feel free to check it out here.

  • HTML
  • CSS
